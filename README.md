# react-export-csv

## Install

```bash
npm install --save react-export-csv
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-export-csv'
import 'react-export-csv/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
